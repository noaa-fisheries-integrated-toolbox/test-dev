NOAA Fisheries Integrated Toolbox

Tool: Standardized trends metrics for listed PNW salmonids (NWCTrends)

Authors

E.E. Holmes

Description

NWCTrends runs the standardized trends metrics used in the 2015 and 2020 5-year NWFSC PNW Salmonid Viability Report which support the 5-year Status Reviews prepared by WRO.
